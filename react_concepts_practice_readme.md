# React Concepts Practice

A React practice project covering React Hooks, state management, routing, Context API, API fetching, component communication, and reusable component development using Vite and JavaScript.

---

## Features

- React Hooks
- useState Hook
- useEffect Hook
- useContext Hook
- useMemo Hook
- useCallback Hook
- React Router DOM
- Context API
- API Fetching
- Dynamic Routing
- Props Handling
- Parent to Child Communication
- Child to Parent Communication
- Conditional Rendering
- State Management
- Reusable Components
- Performance Optimization

---

## Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/pradeepchandhu/react-concepts-practice.git
```

### Navigate to Project Folder

```bash
cd react-concepts-practice
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
